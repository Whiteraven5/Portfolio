Master Thesis - Automatization of chamfering process using robot technology on charge air cooler/EGR for Vestas Aircoil A/S

Implemented automation for manual chamfering process at Vestas Aircoil A/S, employing cutting-edge technologies and robotics expertise.

Utilized: Linux, ROS, Python, Gazebo, Arduino, Ansys, Machine Learning (YOLO), MATLAB, and Solidworks to develop a complex system.

Achievements:

  • Successfully automated the manual chamfering procedure.
  • Integrated a diverse set of elements, including a UR5e robot manipulator and MiR200 mobile robot, into a cohesive system, and complete control through Python and ROS.
  • Developed an end effector tailored for the task and utilized a real-time object detection (YOLO) for pipe recognition.
  • Conducted FEM analysis to verify the feasibility of the chamfering process.
  • Created a digital twin for virtual testing and experimentation.
  • Demonstrated the feasibility of chamfering through an automated process on a scaled printed cooler prototype.
